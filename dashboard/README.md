# ERGONEKTIM Assessment Room

A bilingual, product-facing reader for completed
`ergonektim.assessment.v1` artifacts.

The assessment room presents the six ERGONEKTIM observers independently,
preserves input/kernel/recertification custody, and exposes the structural
trajectory and supporting evidence. It does not upload the opened artifact,
recompute scientific state, or collapse the observer panel into a global
score.

## Run locally

Prerequisite: Node.js `>=22.13.0`.

```bash
npm ci
npm run dev
```

Use **Open JSON / Abrir JSON** to inspect an assessment generated by the
ERGONEKTIM CLI, or select the included synthetic demonstration. The language
toggle changes presentation only; it never changes the underlying evaluation.

Validate a release with:

```bash
npm test
```

The hosted instance is private by default. Opening an assessment remains a
client-side operation.

# Sala de evaluación ERGONEKTIM

Lector bilingüe orientado a producto para artefactos
`ergonektim.assessment.v1` terminados.

La sala presenta por separado los seis observadores de ERGONEKTIM, conserva la
custodia de entrada/kernel/recertificación y permite inspeccionar la trayectoria
estructural y su evidencia. No transmite el artefacto abierto, no recalcula el
estado científico ni colapsa el panel en una puntuación global.

## Ejecución local

Requisito: Node.js `>=22.13.0`.

```bash
npm ci
npm run dev
```

Use **Abrir JSON / Open JSON** para inspeccionar una evaluación generada por el
CLI de ERGONEKTIM, o seleccione la demostración sintética incluida. El
conmutador de idioma solo cambia la presentación; nunca modifica la evaluación.

Valide una entrega con:

```bash
npm test
```

La instancia alojada es privada de forma predeterminada. La apertura de una
evaluación permanece completamente en el navegador.
